# Changelog

## 0.0.1 — 2026-05-05

### Added

- Three-way merge conflict resolution with Local | Result | Remote panes
- Monaco-based diff editor with synchronized scrolling
- Quick-apply hunk via gutter glyph click (`»` / `«`)
- Auto-merge non-conflicting changes on open
- Conflict navigation with F7 / Shift+F7
- Magic Resolve: auto-resolve whitespace-only and import-only conflicts
- Intra-line word/char diff granularity
- Branch compare with file tree and side-by-side diff
- Editable diff view with hunk-apply and file-revert
- Whitespace ignore modes (trim trailing, collapse runs, ignore all)
- Status bar indicator for conflicted files
- Context menu integration (SCM and Explorer)
- Apache-2.0 license
